Must obtain an API key from Google Cloud Platform for "custom search api" loaded as a system environment variable named "CUSTOM_SEARCH_KEY" in order to run.

CONTROLS:
Click within text box & type desired search term, hit enter to submit. Click away to use other key commands and ensure they work correctly.
left/right on arrow keys cycle through different drawing modes
up/down on arrow keys determines the density of each drawn shape
enter cycles to a new image within the current search term
